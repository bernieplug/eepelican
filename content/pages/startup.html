Welcome to Efficient Era!

We are excited to have you use our software. In order to derive maximum benefit from our ever growing set of tools, we have put together a recommended sequence of steps to follow.

1. Complete MWS Authorization

Our software relies on MWS APIs to gather your data and send emails on your behalf. If you haven't activated your account yet, simply log into Efficient Era and follow the instructions for MWS authorization.

Note: If you are using Efficient Era for Japan only please note that you will need either and Amazon.com or Amazon.co.uk account.

2. Set up Alerts

Efficient Era offers broad range of alerts such as Listing Hijack Alerts, Sales Alerts, Product Review Alerts and more to help you stay on top of your business. This is a list of alerts we support:

You can adjust the frequency of alerts to one that you prefer. By default all of these alerts are set to "off".

There are a few alerts that are enabled by default. These are:

  • Product Review Alerts (daily)
  • Seller Feedback (daily)
  • Question Alerts (weekly)
  • Large Order Alerts (daily)

If you would like to change these frequencies simply access the tool page, and you will be able to find tool specific settings in the top right area.

3. Create Follow-up Email Campaigns (Watch Video)

Emails are a great way to provide customer support and also request for reviews. Your email campaigns can be set to trigger each time a product was purchased, shipped or delivered. Define the layout and formatting of the email as you like. To enable and design Email Campaigns, access Email Automation settings within the Email Automator tool and make sure the follow-up email button is enabled. Also, choose the marketplaces you would like automation to be enabled for.

In order for our Email Automator can start using Amazon's buyer-seller messaging system to send follow-up emails you need to add noreply@efficientera.com to the list of approved senders on Amazon

The steps are:

  • From Seller Central Settings select Messages in the top right area
  • Select Authorized Emails


  • Add noreply@efficientera.com to the list of approved senders

4. Set up Bulk Order Alerts

While you are in the Email Automation settings (same as the previous step), you can specify how you would like to be notified each time there is a bulk order. Specify what you consider a bulk order. We have set it to default value of 10 but you can change this at any time.

5. Modify Product Details

Products are detected automatically from Amazon. By default, the listing title on Amazon is the display name within Efficient Era. However, sometimes these titles can be fairly long when selecting them in dropdowns or settings. Feel free to edit these display names. These will only affect how your product is displayed within Efficient Era. In order to change display names, access Product Settings from the gear icon under your username in the top right corner of any page. You can also upload product details via csv.

Once in the Product Settings page, you can provide us with the landed cost of your ASINs. This will be useful in displaying profit calculations in the Efficient Era dashboard.

Note: Since we auto-detect your products from Amazon, based on a listing report that we request, you might start noticing a new type of daily notification from Amazon letting you know that this report was requested. Since this alert can get noisy for some people, we recommend switching it off by going to Seller Central, access Notifications under your account name and then deselect

6. Product Review Settings (Watch Video)

Access Product Review settings located in the right top area within the Product Review dashboard. Would you like to be notified each time you receive a review? Or would you like one daily digest email instead? By default, you will receive a weekly digest. For negative reviews set the star level below which you want to be notified.

7. Set up Seller Feedback automation & alerts

Our Seller Feedback tool gives you email notifications, dashboard based analytics of your seller performance, as well the ability to send automated mails each time you receive a negative Seller Feedback. Access Automatic Replies and Settings from the Feedback Manager tool.

VAT Invoice Automation (optional) (Read Documentation) (Watch Video)

If you sell in Europe and need to help your customers generate VAT invoices. Provide your VAT registration number and countries in which you are VAT registered.
